# Security Policy

## Supported Versions

This project is pre-release. Security fixes will be applied to the default development branch until versioned releases exist.

## Reporting a Vulnerability

Please do not report security vulnerabilities in public issues.

Instead:

1. Contact the repository owner privately through the GitHub account associated with this repository.
2. Include a clear description of the issue, affected files or flows, and reproduction steps.
3. Provide impact details and any suggested mitigation if known.

## Response Expectations

- Initial acknowledgment target: 5 business days
- Triage target: 10 business days
- Remediation timing: depends on severity and scope
